صيد الثغرات

مسار واضح: نطاق، dedupe، مراجعة معرفة/مصدر، إثبات، ثم تقرير قابل للمراجعة.

تقرير
1. النطاقحدد منصة البرنامج والرابط والأصول المسموحة.
2. Dedupeافحص الجدة قبل كتابة التقرير حتى لا نكرر الإغلاقات السابقة.
3. الدليلاجمع أثرًا قابلًا للعرض: خطوات، لقطة، ملف PoC، أو مسار كود.
4. تقريراكتب تقريرًا بدليل وقرار جاهزية، ثم اعرضه في التقارير.

يعرض التقرير والإثبات داخل فتحية مع قرار جاهزية واضح.
